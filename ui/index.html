Usage

Set the USBMUXD_SOCKET_ADDRESS environment variable in your container:

docker run -e USBMUXD_SOCKET_ADDRESS=host.docker.internal:27015 your-image

For iOS 17.4+ tunnel support, also set:

docker run \ -e USBMUXD_SOCKET_ADDRESS=host.docker.internal:27015 \ -e GO_IOS_AGENT_HOST=host.docker.internal \ -e GO_IOS_AGENT_PORT=60105 \ your-image

Test with go-ios:

docker run --rm \ -e USBMUXD_SOCKET_ADDRESS=host.docker.internal:27015 \ -e GO_IOS_AGENT_HOST=host.docker.internal \ -e GO_IOS_AGENT_PORT=60105 \ ghcr.io/danielpaulus/go-ios:latest list
